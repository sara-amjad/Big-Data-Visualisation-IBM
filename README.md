# 📊 IBM Big Data Analytics and Decision-Making Dashboard

**Author:** Sara Amjad | **Date:** May 2026  

> **Academic Context:**  
This project was developed as part of Unit 05: Big Data and Visualisation under the Pearson BTEC HND in Digital Technologies. The project demonstrates the implementation of Big Data analytics, statistical analysis, data manipulation, dashboard development, and business intelligence visualisation techniques within a simulated IBM business environment.


## 📌 Project Overview
This repository presents a Big Data analytics and business intelligence project designed within a simulated IBM Islamabad environment. The project focuses on transforming raw business data into meaningful insights using statistical analysis, graphical visualisation techniques, Excel dashboards, pivot analysis, and data-driven decision-making methodologies.

The project explores the strategic role of Big Data in organisational decision-making, evaluates analytical and visualisation techniques, and demonstrates practical data manipulation using dashboards, pivot tables, regression analysis, clustering, and business intelligence reporting.


## 🧠 Key Features

- 📈 Big Data analytics and business intelligence reporting  
- 📊 Excel dashboard development and data visualisation  
- 🧮 Regression analysis and clustering techniques  
- 📂 Data cleaning, filtering, grouping, and transformation  
- 📋 Pivot table and pivot chart implementation  
- 🔍 Data mining and A/B testing methodologies  
- ⚖️ Ethical and compliance considerations in data analytics  
- 🏢 Simulated IBM business intelligence environment
